IRC Message data.

Fields

tags

Message tags as defined by IRCv3.2.

prefix

The message prefix (or source) as defined by RFC 2812.

command

The IRC command as defined by RFC 2812.

args

The command arguments.

suffix

The message suffix as defined by RFC 2812. This is the only part of the message that is allowed to contain spaces.
